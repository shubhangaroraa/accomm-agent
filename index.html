Setup guide

Step 1 — Anthropic API key
Go to console.anthropic.com → API Keys → Create key. Paste into sidebar.

Step 2 — Google OAuth token
Go to developers.google.com/oauthplayground → find Google Sheets API v4 → tick https://www.googleapis.com/auth/spreadsheets → Authorize APIs → Exchange authorization code for tokens → copy Access token → paste into sidebar.

Step 3 — Save & run
Click Save keys to browser, then fill in the form and click Run search.

This tool runs via a local Node.js server (server.js) which proxies Anthropic API calls to avoid browser CORS restrictions. Keys are stored in your browser only.
